Objeivos

  • Crear un formulario donde se pueda ingresar un usuario y una contraseña. Realice la respectiva validacion de ingreso (puede utilizar una constante para manejar el usuario valido). Cuando el usuario sea valido mostra un mensaje de fondo verde que diga 'Ingreso exitoso' y cuando falle mostrar un mensaje de fondo rojo 'Usuario y/o Contraseña invalidos'
